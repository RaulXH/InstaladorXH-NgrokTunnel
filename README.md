# Comandos de Instalacion
```
cd $HOME
git clone https://github.com/RaulXH/InstaladorXH-NgrokTunnel/
cd InstaladorXH-NgrokTunnel && bash Ngrok-Tunnel
```



<p> Si tiene problema de conexion, Encienda la Zona Wifi </p>
